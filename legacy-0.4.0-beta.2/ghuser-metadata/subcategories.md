# SubCategories

**01.Modeling:** Tools that perform some typical modeling tasks for use with urban models.

**02.Queries:** Set of tools that carry communication functions with the database management system. Here are found both components that deal with geometry and those that deal with alpha-numeric data. Similarly, there will be read components (of existing data) and write components (of new data to be stored in the database).

**03.Utilities:** Some components that can be used in conjunction with those found in the 02.Queries group to make it easier to get input data already formatted for use.

**04.Dataviz:** A set of components for building data visualizations. As output, they provide custom graphs visible on the rhino screen, and are also able to export them in fully configured svg format for later use in publications or further editing.
